# Customer Intelligence System using Classification, Ensemble Learning and Clustering

## Overview

This project develops a Customer Intelligence System using machine learning techniques to identify meaningful country segments and predict cluster membership. The workflow combines clustering and classification approaches to generate data-driven insights.

## Features

* Data Preprocessing and Cleaning
* Exploratory Data Analysis (EDA)
* Feature Scaling using StandardScaler
* K-Means Clustering
* DBSCAN Clustering
* PCA-Based Cluster Visualization
* Random Forest Classification
* XGBoost Classification
* Hyperparameter Tuning using RandomizedSearchCV
* Model Performance Evaluation
* Overfitting and Underfitting Analysis
* Customer Intelligence Segment Insights

## Machine Learning Techniques

### Clustering

* K-Means
* DBSCAN

### Classification

* Random Forest Classifier
* XGBoost Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Matplotlib
* Seaborn

## Results

The project successfully segmented countries into meaningful clusters and predicted cluster membership using classification models. XGBoost achieved the best overall performance, while clustering techniques provided valuable insights into country-level socio-economic patterns.

## Outcome

The system provides data-driven segmentation and predictive insights that can support decision-making, resource allocation, and strategic planning.
